Para esta entrega, se utilizó la programación "Mobile First", teniendo en cuenta 3 dispositivos:
    1. Telefono celular             480px
    2. Tablet                       768px
    3. Notebook/Escritorio          992px


Se aplico CSS Grid en la sección de Galería.
Se aplico Flexbox en el resto de las secciones.
Se aplico en Galería ligthbox para mejor visualización de imagenes.
Se aplico animación de números en sección de resumen.
Se aplico contador regresivo simulando un sorteo.
Se utilizo normalize para quitar los estilos por defecto agregados por los navegadores.
Se incorporó Leaflet (gratuito) reemplazando a google maps, ya que la api es paga.
